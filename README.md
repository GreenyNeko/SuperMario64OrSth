# Super Mario 64 Or Sth

Super Mario 64 Or Sth is a fangame attempting to remake Super Mario 64 in godot using LibSM64 as a parody/troll. It was created for the purpose of the [troll/laughing level/game contest](https://smwtrolls.com/contest/make-us-laugh-troll-contest).

## Source Code

I want to share the source code in case people are interested in how I managed to make some things work. I, of course, have no idea about how to use LibSM64 so it's all trial and error. :3
Hope this helps people.

## How to Use

You can create a godot project, download or clone [LibSM64-godot](https://github.com/Brawmario/libsm64-godot/) or load it via Godot Engine and then override the files and/or add the files you want. Or copy over the changes I made.
